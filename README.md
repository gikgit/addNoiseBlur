# AddNoiseBlur Algorithm

1. ./compilegcc AddNoiseBlur
2. ./AddNoiseBlur
3. input the size of lowpass filter mask. for example: '5' the larger size, the more blur
4. input the variance of noise, for example: '20' (Gaussien noise)
5. result
